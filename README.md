###### Commands
| Command | Description |
| --- | --- |
| @help | Information about the commands. |
| @cheat | Betting cheat sheet. |
| .hit | Draw a card. |
| .mini | Toggle the mini deck. |
| .play | Start a game. |
| .stand | Stop drawing cards. |
| .stop | End the curent game. |
